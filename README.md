# 🎙️ Desktop AI Voice Assistant

A Voice-Controlled Desktop Assistant built using Python that helps automate everyday tasks through simple voice commands. This assistant can open websites, fetch information, play music, and perform system-level operations — all hands-free.

---

## 🚀 Features

- 🎤 Voice Recognition for command input  
- 🔊 Text-to-Speech responses  
- 🌐 Open websites (Google, YouTube, etc.)  
- 🔍 Perform Google searches  
- 📖 Fetch Wikipedia summaries  
- 🌦️ Get real-time weather updates  
- 🎵 Play music from system  
- 📸 Capture images using webcam  
- ⚙️ Execute system commands  

---

## 🛠️ Tech Stack

- Python  
- SpeechRecognition  
- pyttsx3 (Text-to-Speech)  
- OpenCV (Webcam access)  
- Wikipedia API  
- Webbrowser module  
- Requests (for APIs)  

---
## ▶️ Usage

Run the assistant:
->python main.py
Then start giving voice commands like:-
"Open YouTube"
"Search Python programming"
"What is AI?"
"Play music"
"Take a photo"
